# STM32 Microcontroller Labs (Keil µVision5)

This repository contains laboratory works focused on programming the **STM32F200 microcontroller** using **Keil µVision5**.

The labs demonstrate core embedded systems concepts including GPIO control, interrupts, timers, and signal analysis using an oscilloscope.

---

## 📚 Course Information

**Course:** Microcontroller Programming  
**Semester:** 2nd Year – 2nd Semester (2026)  
**University:** Peter the Great St. Petersburg Polytechnic University  

---

## 📖 Overview

The objective of these labs is to gain practical experience in:
- Low-level (register-based) programming  
- GPIO configuration and control  
- Interrupt handling (EXTI, NVIC)  
- Timer-based operations  
- Signal measurement using an oscilloscope  

---

## 🧪 Lab Works

### 🔹 Lab 1 – GPIO & Project Setup
- Creating a project in Keil µVision5  
- Direct register manipulation (without libraries)  
- Controlling LEDs via GPIO  
- Implementing delays using loops  

---

### 🔹 Lab 2 – Signal Measurement
- Using STM32 as a signal generator  
- Measuring signal using an oscilloscope  
- Analyzing amplitude, frequency, period, and duty cycle  

---

### 🔹 Lab 3 – Interrupts (EXTI & NVIC)
- Handling external interrupts  
- Configuring interrupt priorities  
- Button-triggered LED control  
- Working with EXTI lines and NVIC  

---

### 🔹 Lab 4 – Timers & Low Power Mode
- Using SysTick timer for precise delays  
- Generating periodic signals  
- Implementing sleep (low power) mode  
- Wake-up using external interrupts  

---
